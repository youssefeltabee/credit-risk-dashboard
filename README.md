# Credit Risk Predictor Dashboard

Interactive Streamlit dashboard for credit risk assessment using XGBoost with SHAP explainability.

**Data:** 50,000 loan applications | **Model:** XGBoost (PR-AUC 0.803) | **Stack:** Python, Streamlit, Plotly
